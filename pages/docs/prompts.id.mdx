
# Pusat Prompt

import PromptFiles from 'components/PromptFiles'

Pusat Prompt adalah kumpulan prompt (petunjuk atau instruksi) yang berguna untuk menguji kemampuan LLM (Model Bahasa Besar) dalam berbagai tugas dasar dan kompleks. Kami berharap Pusat Prompt ini dapat membantu Anda menemukan cara-cara menarik untuk memanfaatkan, bereksperimen, dan membangun dengan LLM.

Bayangkan Pusat Prompt seperti kotak peralatan yang berisi berbagai alat untuk menguji dan mengoptimalkan kinerja "otak" komputer yang sangat canggih (LLM). Setiap prompt di sini adalah seperti kunci yang membuka kemampuan tertentu dari LLM, mulai dari tugas sederhana hingga yang rumit.

Misalnya, ada prompt yang bisa meminta LLM untuk menjelaskan konsep rumit dengan cara yang mudah dipahami anak-anak, atau prompt yang menguji kemampuan LLM dalam memecahkan masalah matematika yang kompleks.

Kami sangat mendorong dan menyambut kontribusi dari komunitas peneliti dan pengembang AI. Jika Anda memiliki ide untuk prompt baru atau cara pengujian yang menarik, jangan ragu untuk berbagi! Ini seperti mengundang teman-teman untuk bermain dan bereksperimen bersama di taman bermain teknologi AI.
