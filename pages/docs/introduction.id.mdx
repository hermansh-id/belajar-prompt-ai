Rekayasa prompt (prompt engineering) adalah bidang ilmu yang relatif baru. Bidang ini fokus pada pengembangan dan optimalisasi prompt untuk menggunakan dan membangun aplikasi dengan model bahasa besar (Large Language Models atau LLMs) dalam berbagai kasus penggunaan.

Keterampilan rekayasa prompt membantu kita memahami kemampuan dan batasan LLMs dengan lebih baik. Bayangkan LLMs seperti asisten pintar yang perlu diarahkan dengan tepat. Para peneliti menggunakan rekayasa prompt untuk meningkatkan keamanan dan kemampuan LLMs dalam berbagai tugas, mulai dari menjawab pertanyaan sederhana hingga penalaran aritmatika yang rumit. Sementara itu, para pengembang menggunakan rekayasa prompt untuk merancang teknik prompt yang kuat dan efektif, yang berfungsi sebagai jembatan antara LLMs dan alat-alat lainnya.

Panduan komprehensif ini mencakup teori dan aspek praktis rekayasa prompt. Kami akan membahas bagaimana memanfaatkan teknik-teknik prompt terbaik untuk berinteraksi dan membangun aplikasi dengan LLMs. Anggap saja ini sebagai peta jalan Anda dalam menjelajahi dunia rekayasa prompt!

Semua contoh dalam panduan ini telah diuji menggunakan model `gpt-3.5-turbo` di [OpenAI's Playground](https://platform.openai.com/playground), kecuali dinyatakan lain. Model ini menggunakan konfigurasi default, yaitu `temperature=1` dan `top_p=1`. Prompt-prompt yang disajikan juga seharusnya berfungsi dengan model-model lain yang memiliki kemampuan serupa dengan `gpt-3.5-turbo`, meskipun respons model mungkin bervariasi.
