
# Prompt Adversarial pada LLM

import ContentFileNames from 'components/ContentFileNames'

Bagian ini berisi kumpulan prompt yang bertujuan untuk meningkatkan kesadaran tentang berbagai kerentanan pada LLM (Large Language Models atau Model Bahasa Besar).

LLM adalah sistem kecerdasan buatan yang mampu memahami dan menghasilkan teks seperti manusia. Namun, seperti teknologi lainnya, LLM juga memiliki kelemahan. Prompt adversarial adalah cara untuk menguji dan mengungkap kelemahan-kelemahan ini.

Bayangkan LLM seperti seorang siswa yang sangat pintar. Prompt adversarial seperti pertanyaan jebakan yang diberikan untuk melihat apakah siswa tersebut benar-benar memahami materi atau hanya menghafal tanpa pemahaman mendalam.

Dengan mempelajari prompt adversarial, kita bisa lebih memahami batasan-batasan LLM dan bagaimana cara menggunakannya dengan lebih aman dan efektif.
