
# Menjawab Pertanyaan dengan LLM

import ContentFileNames from 'components/ContentFileNames'

Bagian ini berisi kumpulan prompt (petunjuk) untuk menguji kemampuan LLM (Model Bahasa Besar) dalam menjawab pertanyaan.

LLM adalah sistem kecerdasan buatan yang mampu memahami dan menghasilkan teks seperti manusia. Bayangkan LLM sebagai "otak digital" yang bisa diajak bercakap-cakap dan dimintai bantuan untuk berbagai tugas bahasa.

Prompt adalah instruksi atau pertanyaan yang kita berikan kepada LLM untuk mendapatkan respons yang diinginkan. Misalnya, kita bisa memberi prompt "Jelaskan fotosintesis dengan bahasa sederhana" untuk menguji kemampuan LLM dalam menjelaskan konsep ilmiah.

Dalam konteks menjawab pertanyaan, kita menggunakan prompt khusus untuk melihat seberapa baik LLM dapat memberikan jawaban yang akurat, relevan, dan informatif atas berbagai jenis pertanyaan.
