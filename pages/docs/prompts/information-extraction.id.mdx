
# Ekstraksi Informasi dengan LLM

import ContentFileNames from 'components/ContentFileNames'

Bagian ini berisi kumpulan prompt untuk menjelajahi kemampuan ekstraksi informasi dari LLM (Large Language Models atau Model Bahasa Besar).

LLM adalah sistem kecerdasan buatan yang mampu memahami dan menghasilkan teks seperti manusia. Salah satu kemampuan utama LLM adalah mengekstrak informasi penting dari teks yang diberikan. Ini seperti kemampuan komputer untuk "membaca" dan "memahami" teks, lalu mengambil fakta-fakta penting darinya.

Prompt dalam konteks ini adalah instruksi atau pertanyaan yang kita berikan kepada LLM untuk mendapatkan hasil yang diinginkan. Dengan prompt yang tepat, kita bisa meminta LLM untuk mencari dan mengekstrak informasi tertentu dari sebuah teks.

Misalnya, jika kita memiliki artikel berita tentang peluncuran produk baru, kita bisa menggunakan prompt untuk meminta LLM mengekstrak informasi seperti nama produk, tanggal peluncuran, dan fitur-fitur utamanya.

Koleksi prompt dalam bagian ini akan membantu Anda memahami dan bereksperimen dengan berbagai cara untuk mengekstrak informasi menggunakan LLM. Ini sangat berguna untuk berbagai aplikasi, seperti analisis teks otomatis, ringkasan dokumen, atau pengambilan data dari sumber-sumber yang tidak terstruktur.
