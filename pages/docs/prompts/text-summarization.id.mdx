
# Peringkasan Teks dengan LLM

import ContentFileNames from 'components/ContentFileNames'

Bagian ini berisi kumpulan prompt untuk menjelajahi kemampuan peringkasan teks dari LLM (Large Language Models atau Model Bahasa Besar).

LLM adalah sistem kecerdasan buatan yang mampu memahami dan menghasilkan teks seperti manusia. Salah satu kemampuan menarik dari LLM adalah meringkas teks panjang menjadi versi yang lebih singkat namun tetap informatif.

Peringkasan teks sangat berguna dalam berbagai situasi, seperti:
- Membuat ringkasan artikel berita
- Menyederhanakan dokumen panjang
- Mengekstrak poin-poin penting dari sebuah teks

Dengan menggunakan prompt yang tepat, kita dapat mengarahkan LLM untuk menghasilkan ringkasan yang sesuai dengan kebutuhan kita. Prompt adalah instruksi atau pertanyaan yang kita berikan kepada LLM untuk mendapatkan hasil yang diinginkan.

Di bawah ini, Anda akan menemukan daftar file yang berisi berbagai contoh prompt untuk peringkasan teks. Anda dapat mempelajari dan mencoba prompt-prompt ini untuk memahami bagaimana cara kerja peringkasan teks dengan LLM.
