
# Pemahaman Matematika dengan LLM

import ContentFileNames from 'components/ContentFileNames'

Bagian ini berisi kumpulan prompt (perintah atau pertanyaan) untuk menguji kemampuan matematika LLM (Large Language Models atau Model Bahasa Besar).

LLM adalah sistem kecerdasan buatan yang mampu memahami dan menghasilkan teks seperti manusia. Dalam konteks ini, kita ingin melihat seberapa baik LLM dapat menangani konsep dan perhitungan matematika.

Prompt-prompt ini dirancang untuk menguji berbagai aspek pemahaman matematika, mulai dari operasi dasar hingga konsep yang lebih kompleks. Dengan menggunakan prompt-prompt ini, kita dapat mempelajari kekuatan dan keterbatasan LLM dalam bidang matematika.

Jika Anda baru mengenal dunia LLM dan prompt engineering (teknik merancang perintah untuk AI), koleksi prompt ini bisa menjadi titik awal yang bagus untuk memahami bagaimana kita dapat berinteraksi dengan AI dalam konteks matematika.
