
# LLM untuk Klasifikasi

import ContentFileNames from 'components/ContentFileNames'

Bagian ini berisi kumpulan prompt untuk menguji kemampuan klasifikasi dari LLM (Large Language Models atau Model Bahasa Besar).

LLM adalah sistem kecerdasan buatan yang mampu memahami dan menghasilkan teks seperti manusia. Salah satu kemampuan LLM adalah melakukan klasifikasi, yaitu mengelompokkan teks ke dalam kategori-kategori tertentu.

Prompt adalah instruksi atau pertanyaan yang kita berikan kepada LLM untuk mendapatkan respons yang diinginkan. Dalam konteks klasifikasi, prompt digunakan untuk meminta LLM mengategorikan teks atau informasi yang diberikan.

Dengan menggunakan prompt-prompt dalam bagian ini, Anda dapat mempelajari bagaimana LLM melakukan klasifikasi dan bagaimana merancang prompt yang efektif untuk tugas-tugas klasifikasi.
