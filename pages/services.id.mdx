
# Layanan Kami

## Pelatihan Profesional
Kami menyediakan pelatihan profesional untuk organisasi dan startup untuk melatih tenaga kerja mereka dalam rekayasa prompt (teknik membuat perintah untuk AI), membangun aplikasi dengan model bahasa besar (LLM), dan memanfaatkan AI Generatif untuk bisnis.

Pelatihan kami mengajarkan cara menggunakan LLM dan AI Generatif secara efisien dan efektif untuk keperluan bisnis. Kami membahas teknik-teknik prompt terbaik dan terbaru yang dapat Anda terapkan pada berbagai kasus penggunaan, mulai dari membuat peringkas artikel panjang hingga pendeteksi injeksi prompt, bahkan sampai evaluator berbasis LLM. Tujuannya adalah agar Anda belajar menerapkan teknik prompt lanjutan untuk membantu Anda membangun aplikasi dan produk canggih berbasis LLM secara efektif, serta menggunakannya untuk pengembangan profesional.

Topik-topik yang kami sediakan dalam pelatihan:

- Taksonomi Teknik Prompt (cara mengkategorikan berbagai jenis perintah untuk AI)
- Taktik untuk Meningkatkan Keandalan
- Menyusun Output LLM
- Prompting Tanpa Contoh (Zero-shot Prompting)
- Pembelajaran Dalam Konteks dengan Sedikit Contoh (Few-shot In-Context Learning)
- Prompting Rantai Pemikiran (Chain of Thought Prompting)
- Refleksi Diri & Konsistensi Diri
- ReAcT (Reasoning and Acting, teknik penalaran dan bertindak)
- Generasi dengan Augmentasi Pengambilan (Retrieval Augmented Generation)
- Fine-Tuning & RLHF (Reinforcement Learning from Human Feedback)
- Pemanggilan Fungsi (Function Calling)
- Keamanan AI & Moderasi
- Agen Berbasis LLM
- Evaluasi LLM
- Prompting Adversarial (Jailbreaking dan Injeksi Prompt)
- Menilai LLM
- Kasus Penggunaan LLM yang Umum di Dunia Nyata
